周迅刘璇称与深圳市长许宗衡案无关(图)

深圳市长许宗衡。中新社发 陈文 摄(资料图)

中新网6月10日电 某网站援引香港媒体消息称，深圳市长许宗衡被中纪委调查事件昨日终于获得官方证实。据悉，中纪委调查此案已一年多，突破口来自一名获香港居民身分不久的内地著名女星，此人同许宗衡关系密切。对此，引发网友猜测的内地影星周迅与前奥运会冠军刘璇纷纷表示，此女星并非自己。

人民网转载香港明报的报道称，深圳市长许宗衡被中纪委调查事件获得官方证实，新华社引述中纪委负责人说话称，许宗衡“涉嫌严重违纪，目前正在接受组织调查”。许案将引发深圳官场“大地震”，除女副市长闫小培外，还有两名副市长也接受调查。据了解，中纪委调查组去年奥运会前已开始调查许宗衡受贿问题，一直未有进展，其间许宗衡多次上京活动，为自己解套。但许案的突破口是来自一名港人甚为熟悉的、年前以“优才计划”取得香港居民身分的内地着名女星。然而该女星在一些行为上过于张扬，很快被中纪委及国安人员盯上。据该名被扣查的女星交代，许宗衡给了她不少金钱上的好处，而许在港的“私人事务”也全由她打理，她还帮助过许宗衡亮相央视节目，以增加知名度。

消息发出后，引起网友热议。据媒体总结，近年来以“优才计划”取得香港居民身份的女明星有周迅、汤唯、章子怡、刘璇、黄奕等多人，究竟谁是其中的张扬女明星呢？

对于坊间的猜测，奥运冠军、后来转战娱乐圈的刘璇所在的经纪公司发布声明称：“作为一名奥运冠军，体操运动员刘璇一直热心公益事业、工作努力认真，性情纯良真诚，绝对不会参与有违社会法纪的事情。”

刘璇本人也在自己的博客中发文回击了传言并表达了无奈的心情。在题为“来说一声，都是无稽之谈，还有刚才得奖了”的博客文章中，她写道，“已经不是头一次遇到这样的事情，心里除了莫名其妙的委屈，还有万分的无奈。”她还表示，“清者自清，事情很快就会清楚，也很快就会过去，相信法律一定会给一个公正的结果。”

周迅年前刚刚获准拿到香港身份，也成为网友猜测的对象。周迅经纪人对此予以否认，称周迅根本不认识许宗衡，更谈不上交往过甚。
